Projetos

Ponto Eletrônico

O Ponto Eletrônico Web é um sistema simples e funcional, desenvolvido em uma semana para registrar e gerenciar o ponto de colaboradores. Utiliza PHP, HTML e CSS, com integração ao banco de dados MySQL via phpMyAdmin e ambiente WampServer. Permite cadastro de usuários com papéis de administrador ou colaborador, cada um com interface personalizada. Os dados são armazenados no banco, possibilitando a geração de relatórios de forma prática e eficiente.
